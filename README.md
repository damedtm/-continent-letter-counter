I built this program because I was curious on Web Scraping, Python and Object Oriented Programming. It's currently unfinished and a work in progress.
A problem I ran into was my automatted requests getting blocked by the website. Which in my attempt to work around strengthened my foundational understanding of headers, and network policies used by companies and the owners of websites.
It built on my knowledge of importing python modules for functionality.
I used BeautifulSoup as the parsing library and lxml as the underlying parser. 
The program solves a simple question I had pondered, 'What is the most recurring letter in a continent'. I hope you enjoy going through my work
